# 🗡️ Portfólio — Nicolas Gomes Ruiz

> Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro, inspirado na estética sombria e atmosférica de **Hollow Knight**.

---

## ✨ Sobre

Este repositório contém meu portfólio pessoal, criado para apresentar minha trajetória como estudante de **Tecnologia em Desenvolvimento de Software Multiplataforma** na **FATEC**.

O design foi construído do zero, sem frameworks, com foco em identidade visual marcante e experiência imersiva.

---

## 🎨 Design

A paleta e a atmosfera são inspiradas em **Hollow Knight**:

| Token | Cor | Uso |
|-------|-----|-----|
| Soul | `#c0d4f8` | Cursor, destaques |
| Dream | `#8878c8` | Hover, bordas ativas |
| Gold | `#c8a040` | Labels, ornamentos |
| Void | `#08080f` | Fundo principal |

**Tipografia:**
- `Cinzel` — títulos e monograma
- `Crimson Text` — corpo de texto
- `JetBrains Mono` — labels, tags e código

---

## ⚙️ Tecnologias

- HTML5
- CSS3 (variáveis, animações, keyframes)
- JavaScript puro (Canvas API, IntersectionObserver)

---

## 🪲 Funcionalidades

- Cursor customizado com anel de rastreamento suavizado
- Partículas de "alma" animadas via Canvas 2D
- Atmosfera com gradientes radiais e grain animado
- Selo SVG central com animação de rotação e pulso
- Fade-in por scroll via IntersectionObserver
- Layout responsivo (mobile-first)
